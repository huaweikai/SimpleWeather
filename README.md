# Weather

#### 介绍
是由卡片化的简易天气
内置定位，搜索位置，并能够将天气数据详情展出

#### 软件架构
软件大部分都是MVVM设计，并且由于初步设计，是由很多recycleView嵌套而成，但整体能将效果展示。
